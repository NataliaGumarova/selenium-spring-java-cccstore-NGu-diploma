## 1. Установка JDK
Java Development Kit (JDK) — это набор инструментов, необходимых для разработки и выполнения программ на языке Java. Он включает в себя компилятор, виртуальную машину (JVM) и библиотеки, которые обеспечивают создание, компиляцию и запуск Java-приложений. Установка JDK необходима для того, чтобы разработчик мог писать и тестировать код на Java.
Современным и рекомендуемым источником для скачивания JDK является проект `Eclipse Adoptium`, предоставляющий сборки под названием `Temurin`. Этот дистрибутив является бесплатным, открытым и сертифицированным по стандарту Java SE, что гарантирует совместимость и качество. В отличие от Oracle JDK, Temurin свободно доступен для любых целей, включая коммерческое использование, и поддерживается сообществом и Eclipse Foundation.
Для загрузки JDK с сайта Adoptium (https://adoptium.net/temurin/releases) следует выбрать необходимую версию Java, пакет в зависимости от операционной системы и тип JDK, после чего скачать и установить дистрибутив согласно инструкциям для вашей платформы. После установки рекомендуется настроить переменные среды, такие как PATH и JAVA_HOME, чтобы операционная система и инструменты разработки могли корректно обнаружить JDK.
Использование Adoptium Temurin JDK обеспечивает разработчикам высококачественные и надежные средства разработки Java-приложений с регулярными обновлениями и поддержкой различных операционных систем. Это делает данный дистрибутив удобным и современным выбором для программирования на Java.
* Download `Java platform JDK 17`
  ```
  
  https://whttps://adoptium.net/temurin/releases
  ```
* Установите Java с настройками по умолчанию
* Настройте переменную окружения «Java home».
  Если у вас Windows, нажмите Win+R, введите sysdm.cpl и перейдите на вкладку «Дополнительно» → «Переменные среды».
  Создайте переменную JAVA_HOME.
  Значение переменной — путь к месту установки Java, например, C:\Program Files\Java\jdk*..**.
  Также нужно добавить в переменную PATH путь к папке bin внутри установленной Java, например, C:\Program Files\Java\jdk*..**\bin. Для этого нажмите «Изменить» и добавьте указанный путь.

Перезапустите командную строку (cmd) для применения новых переменных.

Проверьте успешную установку, выполнив в командной строке команды:  
   ```
   java –version
   ```
   ```
   javac –version
   ```
Если версии отображаются без ошибок, значит Java установлена и настроена успешно.

Данная настройка является обязательной для разработки на Java 17, а также для запуска и компиляции приложений, написанных на этом языке.

Чтобы установить IntelliJ IDEA, выполните следующие шаги:

## Загрузка Selenium Java Client Driver

Для подключения Selenium WebDriver к вашему Java проекту скачайте Java Client Driver:

1. Перейдите на страницу загрузки Selenium:  
[https://www.selenium.dev/downloads/](https://www.selenium.dev/downloads/).

2. Найдите раздел с драйверами клиентов и выберите Java.

3. Скачайте файл архива с именем, например, "selenium-server-standalone-3.141.59.jar".

4. Распакуйте содержимое архива в удобное место на диске, например, в папку `C:\selenium\`.

5. В вашем проекте подключите все `.jar` файлы из корня и папки `libs` этой распакованной директории.

## Установка драйвера браузера Chrome (ChromeDriver) на Windows

1. Перейдите на официальный сайт для загрузки ChromeDriver:  
[https://googlechromelabs.github.io/chrome-for-testing/](https://googlechromelabs.github.io/chrome-for-testing/).

2. Чтобы выбрать правильную версию, сначала узнайте версию вашего браузера Chrome:
   - Откройте браузер Chrome.
   - В адресной строке введите `chrome://version/` и нажмите Enter.
   - В верхней строке будет указана версия браузера, например, 140.0.xxxx.x.

3. Скачайте версию ChromeDriver, которая соответствует версии вашего браузера (чтобы избежать проблем совместимости).

4. Распакуйте скачанный архив в папку, например `C:\webdrivers\`.

5. Убедитесь, что в этой папке находится файл `chromedriver.exe`.

6. Добавьте путь к папке `C:\webdrivers\` в системную переменную окружения `PATH`:
   - Откройте "Панель управления" → "Система и безопасность" → "Система" → "Дополнительные параметры системы" → "Переменные среды".
   - Найдите переменную `Path` в списке системных переменных, выберите и нажмите "Изменить".
   - Добавьте новую строку с путем `C:\webdrivers\`.
   - Сохраните изменения и закройте все окна.

7. Перезапустите командную строку (cmd) или PowerShell.

8. Проверьте корректность установки, выполнив команду в терминале:  
   ```
   chromedriver --version
   ```
   Если отобразится версия ChromeDriver, значит установка прошла успешно.

## Запуск Selenium Server Standalone и настройка Selenium Grid

Selenium Server Standalone позволяет запускать тесты на разных машинах и браузерах параллельно.

1. Запустите командную строку (cmd).

2. Перейдите в директорию, где находится файл `selenium-server-standalone-3.141.59.jar`.

3. Чтобы запустить центральный сервер (Hub), выполните команду:  
   ```
   java -jar selenium-server-standalone-3.141.59.jar -role hub
   ```
   
4. При успешном старте в консоли вы увидите сообщение:  
   ```
   INFO - Selenium Grid hub is up and running
   ```

5. Чтобы управлять подключенными узлами (Nodes), откройте новый терминал.

6. Запустите узел, указав путь к драйверу Chrome:  
   ```
   java -Dwebdriver.chrome.driver="C:\chromedriver\chromedriver.exe" -jar selenium-server-standalone-3.141.59.jar -role node -hub http://localhost:4444/grid/register/
   ```

7. Перейдите в браузере по адресу:  
   [http://localhost:4444/grid/](http://localhost:4444/grid/)  
   чтобы увидеть панель управления Selenium Grid, где можно увидеть зарегистрированные узлы и доступные браузеры.

8. Узлы могут запускаться на разных компьютерах с разными операционными системами и браузерами. Стандартная конфигурация содержит 5 экземпляров Chrome, 5 Firefox и 1 Internet Explorer.

## Простой тест
Чтобы убедиться,что `Selenium` успешно подключён, вставьте следующий код и скомпилируйте его.
```
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.chrome.ChromeDriver;

public class SimpleSeleniumTest {
    public static void main(String[] args) {
        // Укажите путь к драйверу ChromeDriver, если он не добавлен в PATH
        System.setProperty("webdriver.chrome.driver", "C:\\webdrivers\\chromedriver.exe");

        // Создаём объект драйвера браузера Chrome
        WebDriver driver = new ChromeDriver();

        // Открываем страницу Selenium
        driver.get("https://www.selenium.dev/");

        // Проверяем, что заголовок страницы содержит слово "Selenium"
        String title = driver.getTitle();
        if (title.contains("Selenium")) {
            System.out.println("Тест пройден: заголовок страницы - " + title);
        } else {
            System.out.println("Тест не пройден: заголовок страницы отличается");
        }

        // Закрываем браузер
        driver.quit();
    }
}
```

Этот код:

- Запускает `Chrome` браузер,
- Открывает сайт https://www.selenium.dev/,
- Проверяет заголовок страницы для базовой валидации,
- И закрывает браузер.

Если в консоли появится сообщение «Тест пройден:…», значит Selenium WebDriver и драйвер работают корректно.
